# Politique de securite

Ce projet est un site statique : il ne contient pas de serveur applicatif, pas de base de donnees et pas de stockage distant des saisies utilisateur.

Pour signaler un probleme de securite, ecrire a :

`chateaunathan@proton.me`

Merci de ne pas ouvrir d'issue publique contenant un detail exploitable avant qu'une correction ait pu etre preparee.
